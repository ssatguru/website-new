Unity3D Island

Cobbled this together using
  • Unity3D Game engine for containing or assembling "game" logic (C#) and other artifacts. Also for creating some items like sea, island, trees, grass, sky etc,
  • Blender for creating animations and 3d structures (table, chair, shed etc)
  • Mixamo for freebie rigged characters (man, dog)
  • Internet for freebie audio files - hindi song, waves, dog bark, birds etc :))

Navigation keys
  • to move left, right, forward or back use left, right, up or down arrow or the "a", "d","w","s" keys
  • press shift with above keys to run
  • press and drag left mouse button to change direction or to look up or down
  • press and drag right mouse button to look around
  • use mouse scroll button to zoom in or out
  • use button on top right corner to go full screen
  • clicking some items displays a menu of things to do, The cursor changes to hand if an item is clickable (currently - dog, chair,radio, telporter)
